##  继承 
B 数据类型拥有 其他数据类型的属性和方法
可以实现代码复用，使代码更贴合现实生活
抽出多个实体共有的属性和方法成一个父类，子类继承父类的 所有 属性和方法，并且可以有自己特有的属性和方法 

###   实现
如果想 结构体B 继承 结构体A ：
添加 结构体A 作为 结构体B 的匿名属性
```go
type A struct {
	...
}
type B struct {
	A					// A 只能有一个
	[其他结构体类型]	// 多重继承。尽量不要使用
	...
}
```
```go
type A struct {
	...
}
type B struct {
	*A
	...
}
```


###   初始化
可以嵌套初始化 



###   使用
* 通过路径访问
```go
B.A.{字段A|方法A}
```

```go
B.(*A).{字段A|方法A}
```

* 直接访问
若多个匿名结构体有相同的属性/方法，在访问这些数据/方法的时候 必须写上路径 
1. 查找 B （就近原则）是否含有字段/方法
2. 查不到就查找匿名属性字段（递归查找）
```go
B.{属性|方法}
```

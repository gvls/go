##  init 
作用：常用来做初始化工作。
一个包中可以有多个`init()` 。
`init()` 不能被显示调用、不能被引用（赋值给函数变量）。
一个包被多处导入，但其`init()` 只执行一次。

###   格式
```go
func init(){
	...
	[flag]
	...
}
```

###   执行先后
1. 进入 导入的导入的导入... 包
2. 执行 `const`  
3. 执行 `var`  
4. 执行 `init()`  

5. 退回 被导入包 
6. 执行 ... ... ...
7. ... 

8. 退回 main包
9. 执行 ... ... ...
10. 执行 `main()`

##  赋值
把右边的值 赋给 左边的变量
右边 ： 变量、表达式、常量
左边 ： 变量

|--------|-----------------------|
| format | description           |
|--------|-----------------------|
| `=`    | 不能在 `if` 中使用    |
|--------|-----------------------|
| `+=`   | c += a  =>  c = c + a |
|--------|-----------------------|
| `-=`   |                       |
|--------|-----------------------|
| `*=`   |                       |
|--------|-----------------------|
| `/=`   |                       |
|--------|-----------------------|
| `%=`   |                       |
|--------|-----------------------|
| `<<=`  |                       |
|--------|-----------------------|
| `>>=`  |                       |
|--------|-----------------------|
| `&=`   |                       |
|--------|-----------------------|
| `^=`   |                       |
|--------|-----------------------|
| `i=`   |                       |
|--------|-----------------------|
因为 编辑器缘故 `i=` 实际是 `|=`

##  定义并初始化

###   指定类型并初始化 
```go
var <名字> <类型> = <值> //一行定义一个
```
```go
var <名字>, <名字>, <名字> <类型> = <值>, <值>, <值> // 一行定义多个相同类型
```
```go
var (
	<名字> <类型> = <值> // 多行定一个多个各种类型
	<名字> <类型> = <值>
	...
)
```



###   不指定类型并初始化
编译器 会自动根据 初始化变量的值 来推断 变量的数据类型
```go
var <名字> = <值> //一行定义一个
```
```go
var <名字>, <名字>, <名字> = <值>, <值>, <值> // 一行定义多个相同类型
```
```go
var (
	<名字> = <值> // 多行定一个多个各种类型
	<名字> = <值>
	...
)
```

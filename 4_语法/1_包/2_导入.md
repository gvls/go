##  导入package
在 定义包的 下面设置导入
当包被导入了，包必须被使用

###   导入格式
如果 被导入的包 是内置标准库的，不需要写 路径
如果 被导入的包 是自定义的，需要写在`$GOPATH/src/` 下的绝对路径

* 一行写法
`import "[路径]<包名>[/<子包名>]"` 

* 多行写法
```go
import (
	"[路径]<包名>[/<子包名>]"
	"[路径]<包名>[/<子包名>]"
	<新的名字> "[路径]<包名>[/<子包名>]"	// 为包起新名字，旧的名字不能再被使用
	_ "[路径]<包名>[/<子包名>]"				// 不导入该包
)
```

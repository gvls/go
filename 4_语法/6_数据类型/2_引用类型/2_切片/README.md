##  切片
动态数组（不是真正意义上） ： 长度自动变化
数组的一个动态窗口( [索引, 索引) )
使用起来 和数组差不多



###   逻辑结构
数组的指针：当元素的数量变化时，其可能指向别的数组，该指针类型不能使用`[xxx]数据元素的数据类型` 
切片的容量 >= 切片的长度
切片的长度：`len(切片)` 
切片的容量：`cap(切片)` 
```shell
type slice struct {			
	array unsafe.Pointer 	// 指向 某个数组 的第一个元素
	len   int 				// 切片的长度：切片中的 元素长度
	cap   int 				// 切片的容量：切片的第一个元素的位置 到数组末尾元素的位置。
}
```

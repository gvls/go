##  复制
###   浅拷贝
只复制长度、容量、数组的**地址** 
新切片指向的数组 还是原切片指向的数组
```go
切片B := 切片A 		// 切片B  <==复制结构体中的字段的值==  切片A
```
```go
func 方法名(s []数据元素的数据类型){	// s 是新变量，指向的数组还是原数组
	...
}
func main(){
	...
	方法名(s)
}
```



###   深拷贝
复制了长度、容量、数组的每一个**元素**
目标切片指向的数组 不是原切片指向的数组
```go
copy(切片B, 切片A)	// 切片B  <==克隆==  切片A
```

* 数组复制规则
切片A指向的数组 从头到尾 复制到 切片B指向的数组，直到切片B指向的数组装不下
若切片B的长度 小于切片A，复制有数据丢失
若切片B的长度 大于切片A，切片B后面的数据保持原值

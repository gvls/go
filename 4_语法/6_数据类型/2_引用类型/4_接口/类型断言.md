##  类型断言 
```shell
	variate									variate
_________________					______________________
|   type  |value|					|     type	   |value|
-----------------	==transform==>	----------------------
|interface| ... |					|接口的实现类型| ... |
-----------------					----------------------
```

###   not check
```go
type A interface{
}

type B struct {
	name string
}

func main(){
	var a A
	var b B = B{"123"}
	a = b

	var bb B
	bb = a.(B)	// if a `可以转换成B类型` , transform is success  otherwise throw error
}
```


###   have check
```go
type A interface{
}

type B struct {
	name string
}

func main(){
	var a A
	var b B = B{"123"}
	a = b

	bb, boo := a.(B)	// if a `可以转换成B类型` , boo == true  otherwise  boo == false
}
```

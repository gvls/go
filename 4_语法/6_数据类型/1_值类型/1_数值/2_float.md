##  float
float = 符号 + 指数 + 尾数
一些值会有精度损失，使用`float64` 比 `float32` 损失得少一些

|----------------------|------------------------|-------------|
| type                 | size                   | description |
|----------------------|------------------------|-------------|
| `float32`            | -3.403E38 ~ 3.403E38   |             |
|----------------------|------------------------|-------------|
| `float64`(默认,推荐) | -1.798E308 ~ 1.798E308 |             |
|----------------------|------------------------|-------------|

###   值
|------------|-------------------------------------------------|
| way        | example                                         |
|------------|-------------------------------------------------|

|------------|-------------------------------------------------|

|------------|-------------------------------------------------|

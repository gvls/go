##  得到元素
方式和访问 数组 差不多
该方式 不能 修改元素的值



###   通过 byte 方式访问
不能正常访问 中文

```shell
逻辑 : "520中"

物理 : [7]byte{53,50,48, 228,184,173}

字符    5   2   0       中
        |   |   |    /  |  \ 
      _________________________
      |   |   |   |   |   |   |
值    | 53| 50| 48|228|184|173|
      |   |   |   |   |   |   |
      -------------------------
索引    0   1   2   3   4   5		数据元素单位 : byte
```

* 访问格式
```go
<变量>[<索引>] 	// 索引 从0开始
```



###   通过 字符 方式访问
可以正常访问 中文

* 转换数据类型为 `[]rune` 
```shell
逻辑 : "520中"

物理 : [4]rune{53,50,48, 20013}

字符       5         2         0          中
           |         |         |          |   
      ___________________________________________
      |         |         |         |           |
值    |    53   |    50   |    48   |   20013   |
      |         |         |         |           |
      -------------------------------------------
索引       0         1         2          3    		数据元素单位 : rune
```

* 使用 range
中文的字符会被当作为一个数据元素

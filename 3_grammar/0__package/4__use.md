## 2 use
We must `import xxx`   before use `variate` and `function` which belong to `xxx package`

### 3  `variate` 
* `variate` 
`<package name>.<variate name>` 

* `method` 
`<package name>.<variate name>.<method name>` 



### 3  `function` 
* `method` 
`<package name>.<function name>` (isn't `<file name>.<function name>`)

##  use
We must `import xxx` before use `variate` and `function` which belong to `xxx package`

|------------|--------------------------------|
| type       | format                         |
|------------|--------------------------------|
| `variate`  | `<package>.<variate>`          |
|------------|--------------------------------|
| `method`   | `<package>.<variate>.<method>` |
|------------|--------------------------------|
| `function` | `<package>.<function>`         |
|            | (isn't `<file>.<function>`)    |
|------------|--------------------------------|

## default initialize
If not evaluation `variate`, go will initialize it by default value
All `variate` will be initialize , include : `array` , `struct` ...
|--------------|---------------------|
| type         | value               |
|--------------|---------------------|
| `int...`     | 0                   |
|--------------|---------------------|
| `float...`   | 0.0                 |
|--------------|---------------------|
| `complex...` | 0                   |
|--------------|---------------------|
| `string`     | ""                  |
|--------------|---------------------|
| `bool`       | false               |
|--------------|---------------------|
| `quote type` | nil(还没有分配空间) |
|--------------|---------------------|
| `func...`    | nil                 |
|--------------|---------------------|
| `error`      | nil                 |
|--------------|---------------------|

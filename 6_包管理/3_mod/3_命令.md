##  命令
需要在`$GOPATH` 中执行

* 初始化
使用`go module` 管理当前项目
创建`go.mod` 
```go
go mod init [path/name]
```

* 下载依赖
保存目录是 `$GOPATH/pkg/mod/cache` 
```go
go mod download
```

* 编辑`go.mod` 
```go
go mod edit ...
```

* 以图的方式显示依赖
```go
go mod graph
```

* 整理依赖
下载需要的
删除不需要的
```go
go mod tidy
```

* 复制依赖到`vender` 
```go
go mod vender
```

* 检查依赖
check dependency and view if change
```go
go mod verify
```

* 解析依赖为什么需要导入
```go
go mod why
```

* 查看所有依赖
```go
go list -m all
```

* 更新稳定版的依赖
```go
go get rsc.io/samler
```

* 更新特定版本的依赖
```go
go list -m -version rsc.io/sampler
go get rsc.io/sampler@v1.3.1
```

* 使用`vendor` 管理
```go
go build -mod=vendor
```

* 下载依赖并安装
下载到`$GOPATH/pkg/mod` 
添加依赖到`go.mod` 
```go
go get -m <package>
```

* 更新所有依赖
```go
go get -u=patch
```
